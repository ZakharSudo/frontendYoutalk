# YouTalk — Блог о психотерапии и самопомощи

## О проекте

Проект представляет собой двухстраничный сайт для блога компании YouTalk, которая публикует материалы о психотерапии, самопомощи, отношениях и личных границах.

##  Технологии

- **HTML5** — семантическая вёрстка
- **CSS3** — стилизация, Flexbox, Grid, медиа-запросы
- **Методология БЭМ** — именование классов
- **Google Fonts** — шрифт Montserrat
- **SVG** — векторные иконки


##  Структура проекта
youtalk-blog/
│
├── index.html # Главная страница (лента блога)
├── article.html # Страница отдельной статьи
├── css/
│ └── style.css # Основные стили (адаптивные)
├── images/ # Папка с изображениями (PNG, SVG)
└── README.md # Документация

##  Страницы сайта

### 1. Блог (`index.html`)
- Шапка с логотипом, навигацией и кнопкой записи
- Блок с тегами для фильтрации статей
- Сетка карточек статей (3 колонки на десктопе)
- Кнопка "Загрузить ещё"
- Подвал с навигацией, формой подписки и копирайтом

### 2. Статья (`article.html`)
- Полный текст статьи с заголовками, параграфами
- Боковая колонка (`<aside>`) с推荐文章 и баннером
- Адаптивное поведение: на планшетах и мобильных устройствах колонка уходит вниз

 Как запустить проект
Локальный запуск
Скачайте или склонируйте репозиторий

Откройте папку с проектом

Дважды кликните на index.html — откроется в браузере

Запуск через VS Code:
Установите расширение Live Server
Нажмите правой кнопкой на index.html
Выберите Open with Live Server

Через PowerShell (если нужно пересоздать)
powershell
cd ~\Desktop
New-Item -Path "youtalk-blog" -ItemType Directory -Force
cd youtalk-blog
New-Item -Path "css", "images" -ItemType Directory -Force
New-Item -Path "index.html", "article.html", "css\style.css" -ItemType File -Force
